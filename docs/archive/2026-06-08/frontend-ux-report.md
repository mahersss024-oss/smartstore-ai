- Pending submit buttons are used on major server-action forms.
- Dashboard and admin loading screens exist.
- Customer chat separates visible customer messages from hidden system events.
- Customer-facing guard failures are repaired by the model when possible instead of showing technical text.
- RTL Arabic, English, and French locales are present.
